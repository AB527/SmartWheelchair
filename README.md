# Smart Wheelchair Control App

This Android app enables versatile control of a smart wheelchair using buttons, Google Voice commands, and gesture recognition via TensorFlow's Teachable Machine. Designed for convenience, it connects via Bluetooth, offering responsive and reliable control.

## Features

- **Button Control:** Intuitive on-screen buttons for easy wheelchair navigation.
- **Voice Control:** Hands-free commands with Google Voice Input for seamless mobility.
- **Gesture Control:** Recognizes hand gestures via TensorFlow (Google's Teachable Machine) for convenient navigation.
- **Bluetooth Connectivity:** Pairs with the wheelchair to provide real-time control.
